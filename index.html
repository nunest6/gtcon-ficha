Changelog — GTCON Fichas Técnicas
Histórico de versões e atualizações do sistema
v2.8.0 18/03/2026 ● Novas Funcionalidades
Aba Reunião
  • Nova aba Reunião entre Cadastro Geral e Dep. Pessoal
  • Formulário com tópico (input), data, horário, link e transcrição/ata
  • Seletor de participantes vinculado aos usuários cadastrados no sistema — dropdown com busca, chips visuais com nome e @usuario
  • Departamentos envolvidos: Contábil · Fiscal · Tributário · DP
  • Campo de participantes bloqueado para não-admins após o primeiro registro — somente admin pode editar
  • Link da reunião vira link clicável nos cards registrados
  • Transcrição/ata exibida em bloco destacado com formatação preservada
  • Reuniões salvas junto à ficha no Supabase via collectData()
v2.9.0 18/03/2026 ● Novas Funcionalidades
Fiscal
  • Perfil Comercial adicionado — Varejista / Atacadista / Ambos / N/A
  • e-Commerce (com campo de plataforma) movido do Cadastro Geral para a aba Fiscal
  • Perguntas sobre emissão de notas: 100% das entradas e 100% das saídas (Sim / Não / Parcial)
  • Card Emissor de Notas simplificado — removidos login, senha e contingência; mantidos apenas volumes médios
Contábil
  • Licitações removido do Cadastro Geral e movido para o Contábil, junto ao Imobilizado
  • Card Imobilizado detalhado: possui bens, laudo de avaliação patrimonial, método de depreciação (Linear / Acelerado / N/A), controle via sistema/planilha e observações
Cadastro Geral
  • Competência e Grupo Econômico lado a lado no mesmo card, no topo do Cadastro
Correções
  • Botões de remoção nas linhas dinâmicas corrigidos — addContato, addPrefeitura e addErro usavam getElementById('${id}') dentro de onclick HTML, o que não funciona em runtime; substituídos por this.closest('.classe').remove()
  • CSS corrigido — tags <style> duplicadas removidas
v2.7.0 18/03/2026 ● Novas Funcionalidades
Aba Tributário
  • Nova aba Tributário adicionada entre Fiscal e Contábil
  • Card de acesso ao módulo Auditor — Cruzamento de Declarações com badge de exercícios já importados
  • Card de cruzamento SPED removido da aba Fiscal — centralizado no Tributário
Módulo Auditor
  • Módulo renomeado de sped-cruzamento.html para auditor.html
  • Função abrirAuditor() substitui abrirSPED() — alias mantido para compatibilidade
Status do Onboarding
  • Status renomeados: Cadastrado (vermelho) · Implantando (amarelo) · Em Estudo (azul) · Operação (verde)
  • Cor vermelha (sel-red) adicionada ao sistema de status chips
  • Ordem reflete o ciclo real de vida do cliente
CSS Enterprise (v3.0)
  • Design system completo reescrito — tokens de sombra, raio e cor semântica
  • Formulários redesenhados com border 1.5px, focus ring navy e hover states
  • Botões com elevação, transições e hierarquia clara
  • Tabelas, modais, toasts, checklist e log com nova linguagem visual
  • Botões do header corrigidos — fundo translúcido no estado padrão
  • Changelog sem emojis
v2.6.0 17/03/2026 ● Refinamentos
Emojis removidos
  • Todos os emojis removidos da interface — botões, títulos de card, panel titles, checkboxes, status chips e banners
