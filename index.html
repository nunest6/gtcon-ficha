📋 Changelog — GTCON Fichas Técnicas
Histórico de versões e atualizações do sistema
v2.4.0 17/03/2026 ● Repaginação Visual
🎨 Identidade Visual da Marca
  • Logo SVG vetorial fiel à marca — "Gtcon" branco #f4f8ff, "BRASIL" azul-lavanda #adb2da, extraído da logo original pixel a pixel
  • Fonte trocada de Sora → Plus Jakarta Sans — mais corporativa e legível; mono de DM Mono → JetBrains Mono
  • Sistema de cores derivado da logo: navy #011957 como cor primária em toda a UI
🖥️ Interface
  • Header: fundo navy com sombra suave; logo SVG nos headers da app e da tela de login
  • Sidebar: fundo navy escuro #000d36; item ativo com borda esquerda #adb2da e texto branco
  • Tabs: aba ativa sublinhada em navy, hover com fundo suave azulado
  • Botões primários (Salvar, Editar): navy em vez de verde/azul genérico
  • Focus ring de inputs: navy em vez de azul elétrico
  • Títulos de card: navy uppercase com peso 700
  • Tela de login: fundo gradiente navy escuro + card glass morphism (fundo translúcido, borda branca 12%, backdrop-filter blur)
v2.3.0 16/03/2026 ● Novas Funcionalidades
🔒 Trava de Edição Colaborativa
  • Fichas sempre abrem em modo visualização — campos desabilitados por padrão
  • Botão ✏️ Editar para entrar no modo edição com verificação de disponibilidade no Supabase
  • Se outro usuário já estiver editando: 🔒 João Silva editando desde 14h32 — botão bloqueado
  • Heartbeat automático a cada 3 min; trava expira em 10 min se o browser fechar sem salvar
  • Trava liberada ao salvar, ao clicar ✕ Sair da edição ou ao fazer logout
  • Nova tabela Supabase: edit_locks
🧾 Particularidades por Tributo
  • Card com 6 abas: PIS/COFINS · IRPJ/CSLL · ISS · ICMS · IPI · DAS
  • Ponto verde na aba quando há conteúdo preenchido
🔧 Ajustes de Interface
  • Botões Próximo/Anterior entre abas removidos — navegação pelas tabs do topo
  • "Vende para cliente final (B2C)?" adicionado em Tributário — Engenharia Fiscal
  • Aba Comercial removida do menu; dados mantidos no banco para compatibilidade
  • Bloco Responsável Comercial movido para Cadastro Geral; bloqueado para todos exceto admin
  • Nome do consultor exibido no chip do cliente ativo no topo
  • Comercial: somente leitura em todas as abas
  • Changelog embutido no código, acessível pelo badge v2.x.x no header

  • Migração automática dos dados existentes via SQL UPDATE
v2.1.0 16/03/2026 ● Melhorias
🗄️ Zero localStorage

  • Log de atividades migrado para Supabase (logs) — histórico centralizado
  • Migração automática na primeira abertura — dados antigos movidos sem perda
